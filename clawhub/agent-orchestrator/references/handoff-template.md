# Handoff Template

## PM -> Architect

```text
基于以下需求分析结果产出技术方案：

[需求摘要]
- ...

请输出：
1. 模块边界
2. 数据模型/数据流
3. 技术风险
4. 最小可实施方案

不要写代码。
```

## Architect -> Developer

```text
基于以下技术方案完成实现：

[技术方案摘要]
- ...

请输出：
1. 需要修改的文件
2. 具体实现
3. 测试或验证结果

按仓库现有风格实现，不做无关重构。
```

## Developer -> Reviewer

```text
对以下实现做代码审查：

[变更摘要]
- ...

请输出：
1. 严重问题
2. 中风险问题
3. 测试缺口
4. 是否可合入
```

## Any Stage -> Project Manager

```text
基于以下阶段结果更新项目进展：

[阶段结果摘要]
- ...

请输出：
1. 当前阶段
2. 已完成项
3. 未完成项
4. 阻塞项
5. 下一步
```
