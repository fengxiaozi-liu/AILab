## 依赖与执行顺序

- **跨 Phase 强制隔离**：有强依赖关联（某任务必须在前置任务结构稳固后才能开始）的任务，不得放入同一 Phase。
- **同级任务极致压平**：同属于一个前置依赖层级的多项无依赖任务，**必须全部合并收敛到同一个 Phase 内**，绝不允许强行打散到多个无关的后续 Phase 中以拖慢实施效率。
- 默认按 Phase 顺序逐阶段串行硬阻塞。
- 执行器（Agent）在进入某个 Phase 时，将全权处理该组内的所有任务包，自行决定一次性并发实施或依次实施。
- 如果 agent 已识别到项目约束，应以对应项目规则为准。
